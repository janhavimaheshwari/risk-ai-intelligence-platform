# 🛡️ RiskPulse AI Intelligence

## Enterprise Financial Risk Intelligence Platform

RiskPulse AI is an AI-powered financial risk monitoring and executive intelligence platform designed to analyze transaction data, identify risk indicators, generate business insights, and support executive decision-making through an interactive dashboard.

The platform combines transaction analytics, vector embeddings, Retrieval-Augmented Generation (RAG), and Llama 3 to provide leadership teams with actionable financial risk intelligence.

## Installation

Install dependencies:

## Running the Application

Generate embeddings:

python vectorize.py

Run risk analytics:

python risk_analysis.py

Launch the Streamlit application:

streamlit run app.py
